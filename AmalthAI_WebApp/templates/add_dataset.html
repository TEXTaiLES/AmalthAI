Dataset Structure

The uploaded .zip file must follow this exact directory structure:

{% if mode == "classification" %} 
DatasetName/
├── train/
│   ├── class_1/
│   │   ├── img1.jpg
│   │   └── ...
│   └── class_2/
│       └── ...
└── val/
    ├── class_1/
    │   ├── imgX.jpg
    │   └── ...
    └── class_2/
        └── ...

Note: You can also upload a flat structure (only class folders inside DatasetName/) and the platform will automatically perform a 70% train / 30% val split.
